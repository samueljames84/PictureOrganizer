JPG images follows the following syntax

YYYYMMDD_HHMMSS SSS_WWHH <MODEL> <EVENT> <INCREMENT>
YYYYMMDD_HHMMSS = Date and Time Standard Syntax
SSS = Size of the image (1st 2 Char represents Megapixels and last digit is fraction) example 12.1MP is represented as 121
WWHH = Is the Width to Height aspect ratio WW=Width ratio and HH Height ratio.
<MODEL> = Cameral Model
<EVENT> = Event Name specified on the GUI (Adam's Marriage or Birthday) optional
<INCREMENT> = If any duplicate file name with different image properties will add a increment number.

For Videos MOV and MP4
YYYYMMDD_HHMMSS <MODEL> is default with
<EVENT>  is optional
